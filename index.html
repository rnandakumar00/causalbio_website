Example analyses
Failure mode · Loss of network controllability

Trametinib in KRAS-mutant pancreatic cancer

Proximal
ERK suppression Complete
AKT via RAS→PI3K Maintained
Distal
Cell-cycle arrest Not achieved
Apoptotic commitment Not achieved

MEK inhibition fully suppresses ERK. Constitutive RAS simultaneously drives PI3K activation via crosstalk, sustaining AKT independently of the MAPK arm. A single-arm inhibitor cannot achieve network control when the upstream oncogene feeds two independent survival pathways. Trametinib failed across multiple PDAC trials despite measurable ERK suppression.

Failure mode · Signaling–execution decoupling

CDK4/6 inhibition in RB1-loss tumors

Proximal
CDK4/6 suppression Complete
RB phosphorylation No substrate
Distal
Cell-cycle arrest (E2F gate) Not achieved
Apoptotic commitment Not achieved

CDK4/6 is fully suppressed. Without RB1, the cell-cycle gate does not exist as a regulatory node — E2F is constitutively active before and after drug exposure. The failure mode is architectural and invisible to pharmacodynamic readouts, which report the drug as working.

Failure mode · Incomplete network control

Sotorasib + SHP2i in KRAS G12C with GAP loss

Proximal
ERK suppression Complete (with SHP2i)
AKT via RTK→IRS1→PI3K Maintained
Distal
Cell-cycle arrest Partial
Apoptotic commitment Not achieved

Adding SHP2i achieves complete ERK collapse. AKT remains sustained via RTK-driven IRS1 — a parallel PI3K input neither drug targets. ERK feedback relief further upregulates IRS1, compounding AKT maintenance. Full MAPK suppression is achievable. Apoptotic commitment is not.
